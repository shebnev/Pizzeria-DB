# Data Generation & Loading Script

Python-программа для генерации и заполнения тестовых данных в базу данных **pizzeria**.

## Назначение

Скрипт используется для:

* наполнения базы данных реалистичными тестовыми данными;
* имитации работы реальной пиццерии;
* подготовки данных для аналитики, представлений и визуализации.

Генерация данных необходима для проверки корректности структуры БД, индексов и SQL-представлений.

## Стек

* **Faker** — генерация реалистичных данных;
* **mysql.connection** — подключение и загрузка данных в MySQL.

## Логика работы скрипта

Скрипт последовательно выполняет следующие шаги:

1. подключение к базе данных;
2. генерация справочных данных (клиенты, курьеры, пиццы);
3. генерация заказов с привязкой ко времени и клиентам;
4. генерация позиций заказов (order_items);
5. сохранение данных в БД с учётом связей и ограничений.

Данные генерируются таким образом, чтобы:

* сохранялась ссылочная целостность;
* даты и время выглядели реалистично;
* можно было строить аналитические отчёты по периодам.

## Использование

Перед запуском необходимо создать структуру БД (таблицы, индексы, представления).
После выполнения БД будет заполнена тестовыми данными и готова к аналитике.

## Результат

После работы программы:

* таблицы, каждую из которых можно посмотреть, содержат согласованные тестовые данные;
* SQL-представления возвращают осмысленные агрегаты;
* данные могут быть напрямую использованы в Power BI и pandas.

## Примечание

Скрипт предназначен для учебных и демонстрационных целей.
Объём данных и временные диапазоны могут быть изменены для тестирования производительности и аналитических сценариев.
